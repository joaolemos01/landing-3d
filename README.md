# Landing Page — Loja 3D (Portfólio)

Landing page profissional desenvolvida do zero para treinar **HTML, CSS e organização de layout moderno**, com foco em **hierarquia visual**, **responsividade real** e **código limpo**.

## 🎯 Objetivo
- Melhorar estrutura semântica (HTML)

- Responsividade mobile-first

- Produtos (grid)
